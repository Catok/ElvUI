[b]v3.45[/b]
*Fix a few filter issues with the new Reminder Module.

[b]Additions[/b]
*Add new module "Reminders". It has its own section in the In-Game config. This module will show warning icons on your screen when you are missing buffs or have buffs when you shouldn't.
*Add option under General section to automatically vendor gray items for lazy people that can't click a button.
*Add Postal add-on skin. To disable you must simply disable the ElvUI mailbox skin.
*40 Man layout can now have role icons and raid debuff icons displayed. They are off by Default.
*Add Restore Defaults buttons to Assist/Tank, Group Frames. In the UnitFrames section.

[b]Changes[/b] 
*Removed some old code that was used to keep ElvUI working with patch 4.2.
*Enable/Disable individual unit frames or unit frame groups should no longer require Reload UI. This change also fixes people disabling a frame and seeing the default Blizzard frame popping up after a reload or restart.
*Action-Bar key bind mode should now accept incoming clicks from most gaming mice.

[b]Fixes[/b]
*Fix issue with some high resolutions scale becoming lower than .64, which would cause the borders to look messed up.
*Fix strata issue on shaman, debuff highlighting and incoming heals were appearing above the Totem bar.
*Fix logic error with the BGMap skin, should now set alphas correctly when moussing in and out of the frame.
*Fix issue with data text getting reset when using the reset anchors button.
*Fix Raid Icon size not adjusting when changing the option.
*Copying a message from a B.Net friend to the chat edit box should no longer display as "[|Kf11|kBNET_NAME|k]"
*Fix issue with restore bar button on Action-Bars module not working more than one time.
